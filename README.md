# EKRON ENTERPRISE

Sistema Completo de Construtora e Incorporadora

## Stack Tecnológico

- **Frontend**: Next.js 15, React 19, TypeScript, TailwindCSS
- **Backend/CMS**: Supabase (PostgreSQL)
- **Autenticação**: Supabase Auth
- **Storage**: Supabase Storage
- **Validação**: Zod
- **UI/UX**: Framer Motion, Lucide Icons
- **Gráficos**: Recharts
- **Forms**: React Hook Form
- **Hospedagem**: Vercel (Frontend), Supabase (Backend)

## Quick Start

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`

## Configuração Supabase

1. Crie uma conta em https://supabase.com
2. Configure as variáveis de ambiente em `.env.local`
3. Execute o script SQL em `supabase/schema.sql`

## Estrutura

```
src/
├── app/              # Next.js App Router
├── components/       # React Components
├── lib/              # Utilities
├── services/         # API Services
├── types/            # TypeScript Types
└── schemas/          # Zod Schemas
```
